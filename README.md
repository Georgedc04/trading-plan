# DC Trades – Trading Journal & Analytics

DC Trades is a trading journal and analytics dashboard built with Next.js, Prisma, and PostgreSQL.

## Features
- Trading Journal
- Equity Curve
- Performance Statistics
- Win Rate Tracking
- Session Performance (London / NY)
- Trade History Table
- Add Trade System
- Secure Login System

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL (Railway)
- Recharts
